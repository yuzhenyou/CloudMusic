### Build Setup


### 安装依赖
npm install

### 运行项目，默认端口：8090，项目服务根据本地ip生成。
npm run dev
### 请自行修改
>config/index.js  <br />
>src/main.js  <br />
>src/store/index.js  <br />
>三项api。
### 启动服务
服务更新请自行到上面api连接clone。
